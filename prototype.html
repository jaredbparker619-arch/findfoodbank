Application Prototype Overview

The FindFoodBank prototype demonstrates how users can quickly locate nearby food banks, view important details, and get directions using a simple and accessible interface.

User Flow

  1. User enters a ZIP code or allows location access
  2. App displays a list of nearby food banks
  3. User selects a food bank to view hours and requirements
  4. User receives directions to the selected location
